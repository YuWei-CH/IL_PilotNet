# IL PilotNet

This repository contains the simulator PilotNet workflow for the CS588
autonomous-vehicle project.

## Layout

- `pilotnet_model.py`: shared PilotNet model definition.
- `for_simulator/`: simulator training, teacher-student data collection notes,
  and inference commands.

Start with:

```text
for_simulator/README.md
```

Large datasets, model checkpoints, local virtual environments, and debug image
dumps are intentionally ignored by git.
